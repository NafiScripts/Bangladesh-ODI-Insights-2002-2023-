# Bangladesh-ODI-Insights-2002-2023-
This project focuses on analyzing the performance of the Bangladesh national cricket team in ODIs from 2002 to 2023. As a cricket enthusiast with a passion for data, I explored and visualized extensive ODI cricket match data using Python and Pandas, uncovering patterns and insights that offer a fresh perspective on Bangladesh’s ODI journey. The datasets were sourced from Kaggle: ODI Men's Cricket Match Data (2002-2023).

Key highlights from my analysis include:

📌 Bangladesh’s winning percentages across different venues
📌 The impact of toss results on Bangladesh’s match outcomes
📌 Performance trends across different seasons
📌 Batting and bowling averages of Bangladesh players
📌 Identification of top-performing Bangladeshi players

I plan to extend this analysis further, focusing on individual player performances, and I’m eager to share these insights with both the cricket and data analysis communities.
